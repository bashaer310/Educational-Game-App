# Educational-Game-App

This is **an educational game application** built with Unity and Firebase. The game helps autistic children improve their cognitive skills through interactive games and engaging activities, guided by a friendly virtual character. It includes core features such as **user profile**, **progress tracking**, and **game management**.


## Features

- **User Management**:
  - **User Registration** - Parents can create an account for their child
  - **Authentication** - Users can login 
  - **Profile Management** - Users can view and update their profile

- **Game Management**:
  - **Virtual Character Interaction** - A virtual character talks to the child and guides them through the games
  - **Interactive Games** - Children can play engaging games, including Missing Part, Sound Matching, and Object Positioning

- **Progress Tracking**:
  - **Progress Monitoring** - Track the child’s progress while completing games
  - **Performance Feedback** - The system records results and provides simple feedback
